# Tech Stack

- **Core Framework:** Next.js 16 (App Router compilation with Turbopack) & React 19.
- **Database Engine:** MongoDB accessed via Mongoose ODM.
- **Authentication:** NextAuth v4 (JWT session strategy, Credentials provider).
- **Styling:** Tailwind CSS v4 + Radix UI primitives (packaged via shadcn/ui).
- **Icons & Charts:** Lucide React and Recharts.
- **Email Delivery:** Resend transactional mailer integration.
- **AI Engine:** Google Gemini (Gemini 2.5 Flash / 1.5 Pro) & Groq (Llama 3.3 70B) providers.
