# Naming Conventions

- **React Components & Files:** PascalCase (e.g. `LoginForm.tsx`, `DashboardHeader.tsx`).
- **Standard Code Files:** camelCase or kebab-case (e.g. `contact-service.ts`, `rate-limit.ts`).
- **Database Models:** Capitalized singular (e.g. `User`, `Contact`, `Deal`, `Task`, `Notification`, `Activity`).
- **TypeScript Interfaces:** Prefix interfaces with `I` where applicable (e.g. `IContact`, `IDeal`).
- **CSS Classes:** lowercase kebab-case (e.g. `btn-primary`, `backdrop-blur-md`).
